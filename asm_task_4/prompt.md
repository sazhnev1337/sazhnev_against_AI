Необходимо выполнлить задания по написанию ассемблерного кода. Код будет исполняться на симуляторе. 
В приложенных файлах будет спкцификация и ISA симулятора.  Если возникнут дополнительные вопросы 
или не хватит информации, сообщи.

Задание1:
Вектора хранятся в памяти в формате OFP8 E4M3. Пройдитесь по парам элементов, перемножьте и накопите 
сумму в `acc` как float16.

Шаблон1:

```
JMP start
vec_a: DB 2.0_o3, 3.0_o3, 1.5_o3
vec_b: DB 1.75_o3, 1.0_o3, 2.5_o3
acc:   DB 0.0_h

start:
    MOV A, vec_a
    MOV B, vec_b
    MOV C, 3
    MOV D, acc
    ; Решение
    HLT
```

Задание 2:
Матрица W [m,n] и вектор x [n,1] хранятся в памяти построчно в формате OFP8 E4M3. Реализуйте 
умножение матрицы на вектор. Результат (float16) запишите в `res`.

Шаблон2:

```
JMP start
m:   DB 2
n:    DB 3
mat: DB 1.0_o3, 2.0_o3, 0.5_o3
     DB 1.5_o3, 1.0_o3, 2.0_o3
vec: DB 2.0_o3, 1.0_o3, 1.5_o3
res: DB 0.0_h, 0.0_h
start:
    MOV A, mat
    MOV B, vec
    MOV C, m
    MOV D, res
    ; Решение
    HLT
```

Задание3:
Входное x хранится в памяти как float16. Аппаратной инструкции для экспоненты нет — придумайте, как 
её посчитать. Результат запишите в `res`.

Шаблон3:

```
JMP start
x:   DB 1.0_h
res: DB 0.0_h
start:
    MOV A, x
    MOV B, res
    FMOV.H FHA, [A]
    ; Решение
    HLT
```
